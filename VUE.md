# VUE实例
## 数据与方法
当 Vue 实例被创建时，它向 Vue 的响应式系统中加入了其 data 对象中能找到的所有的属性。

1. 当这些属性的值发生改变时，视图将会产生“响应”，即匹配更新为新的值。

2. 当实例被创建时 data 中存在的属性才是响应式的。之后创建的都不是响应式属性。
  
3. 如果你知道你会在晚些时候需要一个属性，但是一开始它为空或不存在，那么你仅需要设置一些初始值。例外是使用 Object.freeze()，这会阻止修改现有的属性。

4. 前缀 $，以便与用户定义的属性区分开来

## 实例生命周期钩子
1.  Vue 实例在被创建时都要经过一系列的初始化过程，这个过程中会运行一些叫做生命周期钩子的函数，给了用户在不同阶段添加自己的代码的机会。
	1. created：（）{} 实例被创建之后执行代码
	2. mounted：（）{} 
	3. updated：（）{}
	4. destroyed：（）{}

2. 不要在选项属性或回调上使用箭头函数如：created: () => console.log(this.a)。箭头函数是和父级上下文绑定在一起的，this 不会是如你所预期的 Vue 实例
