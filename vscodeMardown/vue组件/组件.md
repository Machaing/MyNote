# 一、父组件引用子组件

## 1. 子组件

正常vue写法

## 2. 父组件

在scrip标签里需写下

```
import [引入组件名] from './../xxx.vue'

export default{
    ......
    components:{
        '组件标签名':引入组件名
    }
}
```
使用：
```html
<template>
    <div>
        <组件标签名></组件标签名>
    </div>
</template>
```

# 二、父组传数据 给子组件

## 1. 父组件调用子组件标签时,绑定动态属性
    <v-header :title="title"></v-header>

## 2. 在子组件里面通过 props 接收父组件传过来的数据
    export  default{
        data(){
            return{
                title:"xxxxx"  //如果子组件定义了和接收父组件相同名字的参数，会优先渲染 props数据。不建议这样写
            }
        },
        props:['title']
    }

 ## 3. 子组件验证 传输数据合法性   
    https://cn.vuejs.org/v2/guide/components-props.html#Prop-%E9%AA%8C%E8%AF%81

    props:{
        传入数据名: {
        type:
        required: boolen
        default: //若默认值是数组，必须是函数式返回 如：function () {return { message: 'hello' }}
        validator: function (value) { //自定义验证函数
            // 传入值必须是下列字符串的值
            return ['success', 'warning', 'danger'].indexOf(value) !== -1
        }
        }
    }


# 三、父组传方法 给子组件

## 1. 父组件: 在method定义方法
    <v-header :run="run"></v-header>
    ....
    method{
        run(msg){
            alert("我是父组件的方法:"+msg)
        }
    } 

## 2. 子组件: props:['方法名']
```html
<template>
    <div>
        <button @click="run(123)">
    </div>
</template>
```
```
props['run']
```
